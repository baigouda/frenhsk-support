Questions fréquentes (FAQ)

Comment commencer à apprendre ?

Ouvrez l'application, choisissez un niveau de vocabulaire (Débutant, Élémentaire, Intermédiaire ou Avancé), sélectionnez votre pet pixel, puis appuyez sur « Commencer ». Les mots s'afficheront automatiquement à l'écran.

Que signifie « glisser pour nourrir » ?

Quand vous voyez un mot à réviser, glissez vers la gauche. Votre pet pixel « mangera » ce mot et le mémorisera. Les mots nourris apparaissent dans le widget d'écran d'accueil lorsque le mode Marathon est activé.

Comment fonctionne le mode Marathon ?

Activez le Marathon pour répéter les mots appris sur le widget tout au long de la journée. De plus, le Marathon accélère l'incubation des œufs — chaque jour avec le Marathon activé rapproche l'éclosion.

Comment ajouter un widget à l'écran d'accueil ?

Appuyez longuement sur l'écran d'accueil, appuyez sur le bouton « + », recherchez « Apprendre le Chinois Facile », choisissez la taille du widget et ajoutez-le. Le widget affichera les mots appris lorsque le mode Marathon est activé.

Qu'est-ce que les PV ? Pourquoi diminuent-ils ?

Les PV (Points de Vie) représentent la vitalité de votre pet pixel. Si vous n'étudiez pas pendant un moment, les PV diminuent progressivement. Quand les PV atteignent 0, votre pet meurt. Étudiez régulièrement pour le garder en vie !

Mon pet est mort, que puis-je faire ?

Vous pouvez acheter une Lanterne de Résurrection pour ramener votre pet avec 50% de PV. La Lanterne de Résurrection est entièrement optionnelle.

Comment fonctionne le système de badges ?

Il y a 12 niveaux de badges. Nourrissez suffisamment de mots pour pondre un œuf, activez le Marathon pour l'incuber, et quand l'œuf éclot, votre pet évolue et un nouveau badge est débloqué. Collectez les 12 badges pour forger le Badge Cosmique Suprême !

Comment restaurer mes achats ?

Allez dans Réglages > À propos > « Restaurer les achats ». Cela restaurera toutes les Lanternes de Résurrection que vous avez achetées précédemment.
